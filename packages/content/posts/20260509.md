今朝、長らくお世話になった GitHub Copilot を解約し、  
OpenAI Codex を Plus から Pro にアップグレードしました。  

## 経緯

今月、5月4日あたりから OpenAI Codex で何度か上限制限（5時間以内）に達しました。  
そこで GitHub Copilot に切り替え使用すると、5月5日あたりに 上限に達してしまいました。  
GitHub Copilot の GPT-5 mini を使いつつ、OpenAI Codex の上限緩和すればそっちを使う、みたいなことをしていました。  

## 1本に

GitHub Copilot には随分お世話になったのですが、  
数日程度で使用制限に引っかかってしまうのは困りました。  
来月からは、GitHub Copilot の料金体制が変わりトークン消費ベースに変わるらしいです。  

https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/

今困っている私としては、GitHub Copilot を解約し OpenAI Codex のプランを1つ上げました。  
Plus が月20ドルに対して、Pro は月100ドルです。  
別の案として、月20ドル の Plus にしつつ、80ドル をクレジット追加して今月どうなるか見る、  
を考えました。  
まあでも単純計算したら、今月80ドルも足らないかもなと思い、Pro に乗り換えました。  

ちなみに、キャンペーンで 5月31日まで Pro の利用回数が 2倍とのことです。  

https://help.openai.com/en/articles/9793128-about-chatgpt-pro-tiers

> For Pro $100, to celebrate the launch, you’ll get 2x Codex usage through May 31. That means 10x usage instead of the standard 5x.

まあ、6月なったら考え直します。  

## 終わりに

経費として落とせるので、まあいっか。
